# rm-booking-curve-lab

ホテル宿泊部門のレベニューマネジメント向けに、PMSから出力されたデータを基に
ブッキングカーブ（予約曲線）とシンプルな予測値を生成するためのツールです。

## 現時点の目的（v0 系）

- PMSから出力された「時系列データ（宿泊日 × 取得日）」を読み込む
- 宿泊日 × リードタイム（0〜90日）の LT_DATA を作成する
- LT_DATA を元に、曜日別のブッキングカーブを可視化する
- 予測ロジックはまず「3ヶ月移動平均」などの素朴な方法からスタートし、
  将来的に SARIMA などに拡張する

## 想定データ

- Excel ファイル：例）`大国町_時系列データ.xlsx`
- シート名：`202506` のように「YYYYMM」形式
- シート構造（例）：
  - 行0: 各列に「取得日」の Excel シリアル値
  - 行1以降:
    - 列0: 宿泊日（datetime）
    - 列1以降: 各取得日時点のオンハンド室数（整数）

## 今後の拡張予定

- 室数だけでなく「人数」「売上」の LT_DATA 化
- SARIMA などを用いた月次予測（悲観・基準・楽観）
- GUI（メインメニュー）からの操作
- EXE 化して現場 PC で動作させる
